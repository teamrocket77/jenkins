# Local Jenkins Server

## To validate set the following env variables // should also work with nvim

| Variable | Usage |
| -------- | ----- |
| JENKINS_USERNAME | current username|
| JENKINS_TOKEN | the api token that was created for the current user|
| JENKINS_HOST | the url where the token can be used|
| JENKINS_INSECURE | only set if https isn't available|

## to back up contiainer run backup.sh
